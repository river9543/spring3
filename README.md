# Spring3 게시판 프로젝트

## 프로젝트 소개
Spring Boot 기반으로 만든 게시판 웹 애플리케이션입니다.  
회원가입, 로그인, 글 작성, 수정, 삭제 기능을 포함하고 있습니다.

##  사용 기술
- Java 17
- Spring Boot 3
- Spring Security
- Thymeleaf
- JPA (Hibernate)
- H2 / MySQL
- Lombok
- jQuery + AJAX

##  주요 기능
- 회원가입 및 로그인 (Spring Security)
- 질문 목록 조회 / 상세보기
- 질문 작성 / 수정 / 삭제 (권한 체크)
- 페이징 처리
- H2 콘솔 연동
